# Apache BookKeeper (apache-bookkeeper)
Apache BookKeeper is a scalable, fault-tolerant, and low-latency storage service optimized for real-time workloads. It provides a simple log-oriented storage abstraction called ledgers that is used by Apache Pulsar and other distributed systems for reliable log storage.

## APIs

### Apache BookKeeper
BookKeeper provides a Java client API for creating, reading, and managing ledgers (append-only log segments), along with an Admin REST API for cluster management and monitoring.
